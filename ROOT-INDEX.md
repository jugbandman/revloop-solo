# Vault Index

**Purpose:** RevLoop operating system

## Required Sources
These files MUST be read before any deliverable or content is written.
The write gate enforces this mechanically.

- positioning.md
- icp.md
- brand-voice.md

## Context Files (load as needed)
Additional context that supplements required sources. Located in `context/` (solo) or `c-context/` (team).

## Entity Folders (context.md required per entity)
Writing to any file inside an entity folder requires reading that entity's `context.md` first.

## Session State
Session state lives in `.session/` and is ephemeral (auto-cleared on Stop).
Never reference session files in deliverables.
