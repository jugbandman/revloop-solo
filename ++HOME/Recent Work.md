## How to Use This File

- **Each session:** Add a row to "This Week" for every deliverable, import, or significant system change
- **End of week:** Move "This Week" rows to "Previous Weeks", summarize the week
- **Git commits:** Updated by /wrap or /sync-gh, or manually after pushes
- **Who column:** Track which operator produced the work (from USER.md identity)
- **File column:** Link to the actual deliverable. Use relative paths from ++HOME.
